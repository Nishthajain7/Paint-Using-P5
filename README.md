# MS Paint style drawing application using p5.js

The project provides a clean drawing canvas and a color palette for selecting different brush colors.

# Usage
To change the stroke weight (brush size), change the value of the brush size [variable](https://github.com/Nishthajain7/Paint-Using-P5/blob/1ae32acd52ed82e271327a29697797a126e1bc34/sketch.js#L3).
